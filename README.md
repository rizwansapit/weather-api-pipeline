\#Weather API Data Pipeline



This project collects weather data from an External API and Store it into PostgreSQL.



\##Current Progress

* Created the PostgreSQL Database
* Created the 'weather\_data' table
* added the database schema as a SQL file
* Created a Python virtual environment
* Added PostgreSQL connection configuration
* Connected Python to PostgreSQL using Psycopg
* Verified the 'weather\_data' table from python

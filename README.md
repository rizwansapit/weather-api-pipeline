\#Weather API Data Pipeline



This project collects weather data from an External API and Store it into PostgreSQL.



\##Current Progress

* Created the PostgreSQL Database
* Created the 'weather\_data' table
* added the database schema as a SQL file
* Created a Python virtual environment
* Added PostgreSQL connection configuration
* Connected Python to PostgreSQL using Psycopg
* Verified the 'weather\_data' table from python
* Transform the API arrays into individual weather records
* Converted API timestamps into timezone using python datetime values
* Validated that all API arrays have watching lengths
* Loaded transform weather data records into PostgreSQL
* verified the inserted records using SQL queries



\## Planned Pipeline

Open weather API > Python Extraction > Data Transformation > PostgreSQL weather\_data table
